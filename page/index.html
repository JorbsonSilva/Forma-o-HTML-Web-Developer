Oque é HTML?

Voltar

HTML é a abreviação de HyperText Markup Language(Linguagem de Marcação de Hipertexto), foi criado em no inicio da decada de 1990 pelo pelo físico Tim Berners-Lee e seus colegas. Em 2000 se tornou norma internacional apos W3C se integrar ao desenvolvimento da linguagem na versão 3.2 no ano de 1997.

HTML é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.

Estrutura

Voltar

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da Web. A marcação HTML inclui "elementos" especiais, como head, title, body, header, footer, p, div, img, nav, output, ul, ol, li e muitos outros.

Um elemento HTML é separado de outro texto em um documento por "tags", que consistem no nome do elemento entre "<" e ">".

TAGs

Voltar
  • head - providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
  • title - define o título do documento, mostrado na barra de título de um navegador ou na aba da página.
  • body - representa o conteúdo de um documento HTML.
  • img - representa a inserção de imagem no documento.
  • header - representa um grupo de suporte introdutório ou navegacional.
  • footer - rodapé.
  • p - representa um parágrafo.
  • div - é um container genérico para conteúdo de fluxo, que de certa forma não representa nada.
  • nav - representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação.
  • output - é um elemento no qual um site ou aplicativo pode injetar os resultados de um cálculo ou o resultado de uma ação do usuário.
  • h1 a h6 - representam seis níveis de título de seção.

Atributos

Voltar

Atributos no HTML são palavras-chave usadas dentro da tag de abertura de um elemento para fornecer informações adicionais sobre ele ou para modificar seu comportamento e funcionalidade. Eles consistem em um nome e um valor (como nome="valor") e são aplicados na tag para configurar o elemento de maneira mais específica.

List

Voltar

No HTML pode ser feitas listas ordenadas ou desordenasdas mas que precisa utilizar tags especificas.

  • ul - representa uma lista de itens sem ordem rígida.
  • ol - representa uma lista de itens ordenados.
  • li - é usado para representar um item que faz parte de uma lista.

Link

Voltar

Um link (ou hiperlink) em HTML é um elemento clicável (texto, imagem, etc.) que permite aos usuários navegar entre diferentes páginas da web, documentos ou seções dentro da mesma página. Ele é criado usando a tag "a" (âncora) e seu atributo href, que especifica o endereço do destino. Os links são fundamentais para a navegação na internet.
